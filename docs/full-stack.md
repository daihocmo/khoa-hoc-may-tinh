# Lộ trình tự học Fullstack 6 tháng theo tuần.
Hiện tại lộ trình này mình viết cho cá nhân mình dùng nên nó vẫn chưa được hoàn chỉnh. Mình cũng sẽ chỉnh sửa và cải thiện trong quá trình học luôn.

## Lộ trình học

| **Tuần** | **Nội dung cần học**  | **Nhiệm vụ và mục tiêu** | **Tài nguyên/Dự án**  |
| --- | --- | --- | --- |
| 1  | **HTML, CSS, và DOM**  | - Học HTML structure, forms, và semantics.<br>- Học CSS cơ bản và responsive design.<br>- Tìm hiểu về DOM và JavaScript integration. | - Xây dựng một static webpage.<br>- Add interactivity với DOM manipulation.<br>- [HTML Foundations](https://www.theodinproject.com/paths/foundations/courses/foundations#html-foundations)<br>- [CSS Foundations](https://www.theodinproject.com/paths/foundations/courses/foundations#css-foundations) |
| 2  | **JavaScript cơ bản** | - Học về variables, loops, functions, và arrays.<br>- Tìm hiểu về ES6 features như `let`, `const`, và arrow functions. | - Làm một ứng dụng to-do list.<br>- [30-Days-Of-JavaScript](https://github.com/Asabeneh/30-Days-Of-JavaScript)<br>- [33 Concepts Every JavaScript Developer Should Know](https://github.com/leonardomso/33-js-concepts)  |
| 3 + 4 | **Backend** | - Học cơ bản về stack backend bạn chọn.<br>- Học HTTP cơ bản và request-response cycle.<br>- Tạo REST APIs (sử dụng Node.js và Express). | - Xây dựng một (Node.js) server đơn giản.<br>- Xây dựng một REST API cho một blog platform.<br>- [Fullstack JavaScript](https://www.theodinproject.com/paths/full-stack-javascript)<br>- [Full Stack Ruby on Rails](https://www.theodinproject.com/paths/full-stack-ruby-on-rails)<br>- [Zero To Production In Rust - Bản Sample được đăng trên trang Blog cá nhân của tác giả](https://www.lpalmieri.com/posts/2020-05-24-zero-to-production-0-foreword/) và [Code for "Zero To Production In Rust"](https://github.com/LukeMathWalker/zero-to-production)<br>- [Tất tần tật về học Java](https://github.com/loda-kun/java-all) và [loda-kun/spring-boot-learning](https://github.com/loda-kun/spring-boot-learning)<br>- Còn lại bạn tự tìm ;-; |
| 5  | **NoSQL Databases (MongoDB)**  | - Học NoSQL concepts.<br>- Perform CRUD operations với MongoDB.<br>- Tìm hiểu về schema design và indexing.  | - Xây dựng một database cho một "Bookshelf App" sử dụng MongoDB.|
| 6  | **SQL cơ bản**  | - Học SQL syntax.<br>- Perform CRUD operations với SQL databases (e.g., MySQL hoặc PostgreSQL).| - Xây dựng một simple database cho user data.  |
| 7  | **ORMs (Sequelize/Mongoose)**  | - Học về Object-Relational Mapping.<br>- Sử dụng Sequelize cho SQL và Mongoose cho MongoDB.<br>- Học model relationships.| - Xây dựng một project that integrates Sequelize hoặc Mongoose. |
| 8  | **React cơ bản** (Hoặc Vue, Stelve .etc.) | - Học React nền tảng: JSX, components, props, và state.<br>- Sử dụng React Developer Tools. | - Xây dựng một task tracker app.  |
| 9  | **Styling React Apps** (Hoặc Vue, Stelve .etc.) | - Style React apps sử dụng Tailwind CSS hoặc styled-components.<br>- Học responsive design trong React.| - Restyle your task tracker app.<br>- [Tailwind CSS Docs](https://tailwindcss.com/docs) |
| 10 | **TypeScript cơ bản** | - Học TypeScript syntax và static typing.<br>- Tích hợp TypeScript với React.<br>- Tìm hiểu về type declarations cho APIs và models.  | - Refactor the task tracker app to TypeScript. |
| 11 | **Next.js cơ bản** | - Học server-side rendering (SSR) và static site generation (SSG).<br>- Xây dựng SEO-friendly apps.  | - Xây dựng một blog site sử dụng Next.js.  |
| 12 | **Monorepos và Turborepo** | - Học monorepos và how to organize multiple dự án.<br>- Sử dụng Turborepo cho performance optimization.  | - Tạo một monorepo cho shared libraries trong React và Node.js. |
| 13 | **Websockets & RTC cơ bản** | - Học real-time communication với Websockets.<br>- Học RTC cho video/audio communication.  | - Xây dựng một chat app với Websockets.<br>- Tìm hiểu về RTC cho một video conferencing feature.  |
| 14 | **Testing cơ bản** | - Học unit và integration testing.<br>- Sử dụng Jest và React Testing Library.<br>- Test APIs và frontend components. | - Write tests cho your REST API và React components. |
| 15 | **Authentication & JWT** | - Học secure user authentication.<br>- Sử dụng JWT và cookies cho session management. | - Add login/logout functionality to your blog platform.  |
| 16 | **Tối ưu cơ sở dữ liệu** | - Tìm hiểu về database optimization techniques (e.g., indexing, query optimization).| - Optimize queries cho existing dự án. |
| 17 | **Khái niệm Backend nâng cao**  | - Học thêm về asynchronous programming và error handling.<br>- Học middleware chaining.<br>- Implement các logic backend phức tạp.  | - Refactor APIs với advanced features như rate limiting và complex queries.| 
| 18 | **Ref, Populate, và API Design** | - Sử dụng `ref` và `populate` trong Mongoose cho relationships.<br>- Học API versioning và documentation (Swagger).  | - Add relationships to your MongoDB models.  |
| 19 | **Advanced Frontend Features** | - Học dynamic form handling, animations, và accessibility features trong React.<br>- Tìm hiểu về Framer Motion cho animations. | - Add form validation và animations to the task tracker.|
| 20 | **Project Planning** | - Plan một capstone project.<br>- Define features, Tạo high-level (HLD) và low-level designs (LLD).| - Tạo một project blueprint (e.g., e-commerce app hoặc task tracker).|
| 21 | **Capstone Backend Development**  | - Xây dựng the backend cho your capstone project.<br>- Implement CRUD APIs, authentication, và error handling.| - Set up Express.js, MongoDB, và authentication cho the project. |
| 22 | **Capstone Frontend Development** | - Xây dựng the frontend cho your capstone project.<br>- Sử dụng React hoặc Next.js cho user interface.<br>- Style the app với Tailwind CSS.  | - Tích hợp the backend với the frontend.|
| 23 | **Real-Time Features trong Capstone** | - Add real-time features sử dụng Websockets hoặc RTC.<br>- Implement live chat hoặc real-time notifications.| - Enhance the capstone app với real-time functionality. |
| 24 | **Testing và Deployment**  | - Test the app thoroughly.<br>- Deploy the app to một cloud platform (e.g., Vercel, Heroku).| - Write unit và integration tests.<br>- Deploy the capstone project.  |
| 25 | **Linting và Performance Optimization** | - Set up linting tools (e.g., ESLint, Prettier).<br>- Optimize frontend và backend performance.<br>- Học về caching strategies.| - Add ESLint và Prettier vào dự án của bạn.<br>- Optimize API response times và frontend load times. |
| 26 | **Final Enhancements & Portfolio** | - Polish the capstone project.<br>- Add the project to một portfolio site.<br>- Prepare cho interviews với dự án và một resume. | - Tạo một portfolio cá nhân sử dụng Next.js.<br>- Showcase all 10 dự án trong portfolio của bạn.|

## Danh sách dự án

Danh sách 10 ý tưởng gợi ý để xây dựng ứng dụng Fullstack.

| **Project Name** | **Description**  | **Các bước xây dựng** | **Tech Stack** |
| --- | --- | --- | --- |
| **Task Tracker App** | một React app to track daily tasks với add, edit, và delete functionality.  | 1. Set up React. <br>2. Xây dựng một task list UI. <br>3. Add state management với React hooks. <br>4. Store tasks locally.  | React, CSS |
| **Bookshelf App** | một MongoDB-powered app to manage một collection of books với search và CRUD operations.  | 1. Set up MongoDB.<br>2. Tạo Express APIs cho CRUD operations.<br>3. Connect backend to một React frontend.| Node.js, Express, MongoDB, React|
| **Blog Platform** | một platform cho users to post, edit, và delete blog entries. Includes JWT-based authentication. | 1. Xây dựng REST APIs với Express. <br>2. Sử dụng MongoDB hoặc PostgreSQL cho storing blogs. <br>3. Add authentication.| Node.js, MongoDB/PostgreSQL, React|
| **Chat App** | Real-time chat app sử dụng Websockets với user authentication.| 1. Set up Websockets on the backend. <br>2. Implement chat UI trong React.<br>3. Add JWT authentication.| Node.js, Websockets, React|
| **E-Commerce Store** | một store với product listings, một cart system, và checkout functionality.  | 1. Design product APIs. <br>2. Add cart logic trong React.<br>3. Tích hợp Razorpay/Stripe cho payments.| React, Node.js, MongoDB, Payment APIs |
| **Portfolio Website**  | một personal portfolio to showcase dự án và skills. | 1. Sử dụng Next.js cho SSR.<br>2. Style với Tailwind CSS.<br>3. Add portfolio content dynamically sử dụng JSON.  | Next.js, Tailwind CSS  |
| **Expense Tracker** | một tool to track income và expenses với visualizations.  | 1. Tạo React components cho transactions.<br>2. Store data trong MongoDB.<br>3. Visualize data sử dụng Chart.js. | React, MongoDB, Chart.js  |
| **Online Classroom Platform** | một platform cho faculty to share documents, conduct live sessions, và Tạo assignments. | 1. Xây dựng backend với Express.<br>2. Implement user roles.<br>3. Add WebRTC cho live classes.| Node.js, WebRTC, React |
| **Video Conferencing App** | một basic conferencing app sử dụng WebRTC cho real-time communication. | 1. Set up WebRTC APIs.<br>2. Xây dựng React frontend.<br>3. Manage connections sử dụng signaling servers. | WebRTC, React|
| **Capstone E-Commerce App** | một fully-featured app với real-time notifications, authentication, và performance optimization.| 1. Plan HLD và LLD.<br>2. Develop backend APIs.<br>3. Add Websocket-based notifications.<br>4. Deploy on the cloud. | Node.js, React, MongoDB, Websockets |

### Chi tiết về việc xây dựng dự án

1. **Progressive Complexity**: Projects grow trong difficulty from simple React apps (e.g., Task Tracker) to advanced, multi-featured applications (e.g., Capstone E-Commerce App).
2. **Full Stack Integration**: Emphasizes both frontend và backend skills, including databases, APIs, và real-time communication.
3. **Deployment**: Includes một deployment step (e.g., sử dụng Vercel, Heroku, hoặc AWS) to simulate production environments.
