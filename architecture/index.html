    Chuẩn bị

    Trước khi học khóa này thì nên học C. Nếu định học C thì hãy học cuốn Modern C - Quyển này miễn phí nha. Học C xong sẽ có nền tảng để bắt đầu học OSTEP luôn.

    Trong khóa sẽ có C Bootcamp, nhưng nếu học từ quyển Modern C rồi thì mạnh dạn bỏ qua thôi.
